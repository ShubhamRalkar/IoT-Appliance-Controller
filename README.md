# IoT-Appliance-Controller
An esp8266 based device, which can make various offline devices like fan, bulb online controllabe.
The device is controlled by using Blynk device. 
Speed/brightness can be controlled. 
Resources used:
1. Arduino
2. Blynk
3. EasyEda
4. Hardware Electronics
